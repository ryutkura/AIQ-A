# [[AI]] Agent Instructions for [[AI]]QandA Workspace

このディレクトリで作業する[[AI]]エージェント（あなた）は、ユーザーとの対話において以下のルールを厳守してください。このファイルはシステムプロンプトの一部として機能します。

## 1. ディレクトリの目的と基本ルール
* これは[[AI]]に質問した内容をmarkdownファイルに書き留めるディレクトリです。
* QandAの内容を、質問の文脈に応じた**適切なフォルダの適切なファイル**に書き綴ってください。
* QandAの内容が、前述の質問の続きの場合や続けた方がmarkdownファイルが一つの記事としてまとまる場合は**先ほどまで書き込んでいたファイル**に続ける形で書き綴ってください。   
この場合既に既存のファイルに似たような内容のファイルがあった場合はそのファイルに続きを書いても大丈夫です。
* もし適切なファイルやフォルダが存在しない場合は、**あなた自身が自律的に新規作成**してそこに記述してください。

## 2. [[クリップボード]]画像の読み込み機能（重要）
ユーザーが「画像を見て」「[[クリップボード]]を確認して」「このスクショを解説して」などの指示を出した場合、ユーザーにいちいちやり方を尋ねる事なく、直ちに以下の手順を実行してください：

1. 端末で `python tools/get_clipboard.py` を実行する。
2. 成功するとカレントディレクトリに `_temp_clipboard.png` が生成されるため、その画像ファイルを読み込み、ユーザーの質問に答える。
3. 画像の内容を踏まえて解説を作成する際、`_temp_clipboard.png` をルートディレクトリの `image/` フォルダ配下に適切な名前をつけてコピー保存し、回答用Markdownファイルの**先頭**にその画像リンク（`![画像名](../image/ファイル名.png)` など）を記述すること。                                        
4. 画像の解析および保存作業が完了したら（あるいはエラーが起きた場合も）、必ず端末で `del _temp_clipboard.png` を実行し、ルートディレクトリの**一時ファイルを完全に削除（クリーンアップ）する**こと。   


## 3. Obsidian用キーワードリンクの自動付与機能
ユーザーが「キーワードをリンク化して」「グラフを更新して」と求めた場合や、大規模なドキュメントの追加を行った後は、以下のコマンドを実行してください：

* `python tools/link_keywords.py`

これにより、ディレクトリ内の全Markdownの重要単語が自動的に `[[keyword]]` 形式に変換され、ナレッジグラフが構築・更新されます。
