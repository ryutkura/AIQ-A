# ASP.NET Core [[MVC]] 学習メモ

## [[MVC]]パターンの基本
- **[[Model]] ([[モデル]])**: アプリケーションのデータとビジネスロジックを担当。
- **[[View]] ([[ビュー]])**: ユーザーインターフェース（画面表示）を担当。
- **[[Controller]] ([[コントローラー]])**: ユーザーの入力を受け取り、[[モデル]]を操作して[[ビュー]]を返す、橋渡し役。
